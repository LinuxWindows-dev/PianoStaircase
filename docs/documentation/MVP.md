# Core Elements to Be Created

- Each step plays a musical note
- Some type of sensor to detect when someone steps on a stair
- Remote on/off control system

# Needed to create product:

## Sensors

- To detect when someone steps on a stair

## Sound System

- Each stair triggers a piano note
- Sound played through a speaker
- Needs to be loud enough to hear in a public space but not so loud to annoy professors

## Microcontroller / Central Control

- Reads sensor input
- Triggers appropriate sound
- Handles remote on/off functionality

## Power

- Safe and stable power 
- Does not cause a tripping hazard

## Interface

- Sound can be turned on and off
- Lights can be turned on and off
- Does not get in the way of anything
- Gives credit to creators
